# mvwizr: Mikroverunreinigungen in Schweizer Gewässern darstellen

Tools um Mikroverunreinigungsdaten aus Schweizer Gewässern gemäss den
Bestimmungen der Gewässerschutzverordnung (GSchV) darzustellen.
