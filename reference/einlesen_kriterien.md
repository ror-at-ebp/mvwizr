# Qualitätskriterien einlesen

Liest Qualitätskriterien ein. Bei mehreren Ökotoxikologischen Werten pro
Substanz wird der tiefste Wert gewählt.

## Verwendung

``` r
einlesen_kriterien(kriterien_pfad)
```

## Argumente

- kriterien_pfad:

  Pfad zur Exceldatei mit den Qualitätskriterien

## Rückgabewert

Dataframe mit den Qualitätskriterien
