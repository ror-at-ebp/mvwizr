# VSA ID_Substanz Lookup einlesen

VSA ID_Substanz Lookup einlesen

## Verwendung

``` r
einlesen_vsa_lookup(vsa_lookup_pfad, alle_felder = FALSE)
```

## Argumente

- vsa_lookup_pfad:

  Pfad zur Excel-Datei mit dem ID_Substanz-Lookup des VSA

- alle_felder:

  Alle Spalten exportieren? Standardmässig werden nur die im Weiteren
  benötigten Spalten exportiert.

## Rückgabewert

Dataframe mit Lookups
